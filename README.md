# WGC Taxi Simulator - Backend Service （English）
This project is the backend service for the WGC Taxi Simulator. It is responsible for processing path planning algorithms (WGC algorithm) and providing driver trajectory data to the frontend.

## Tech Stack
Languages: Java (Core Backend) / Python (Algorithm Support)
Framework: Spring Boot
Database: PostgreSQL with PostGIS extension
Deployment: Docker

## Deployment Steps
Java: 17+
Database: Local PostgreSQL instance (Default port: 5432, configurable in application.properties)
Build Tool: Maven

## Development Roadmap
- [x] Static Data API
- [ ] Implement Algorithm Logic
- [ ] Frontend-Backend Integration
